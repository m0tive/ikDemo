CGI Techniques : Inverse Kinematics Demo

Compiling:

The demo is built with Processing (http://processing.org) and uses the 
following libraries:
    * Obsessive Camera Direction (version 1.4) : 
        http://www.gdsstudios.com/processing/libraries/ocd/

    * toxiclibs (version 0018) : http://toxiclibs.org

Controls:

Space - spin camera left
Shift+Space - spin camera right
Arrow keys - Move IK goal in x-z axis
Page Up/Down - Move IK goal in y axis
Esc - Quit
